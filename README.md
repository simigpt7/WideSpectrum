# Wide Spectrum Productions

Premier music production company specializing in music production, mixing, mastering, film scoring, and sound design.

## Tech Stack

- **Frontend**: React 18 + TypeScript + Vite
- **Styling**: Tailwind CSS
- **Database**: Supabase
- **Monitoring**: Sentry
- **Testing**: Vitest, Testing Library, Playwright, axe-playwright
- **Analytics**: Web Vitals

## Getting Started

### Prerequisites

- Node.js 18+
- npm or yarn

### Installation

```bash
npm install
```

### Environment Setup

Copy `.env.example` to `.env` and fill in your credentials:

```bash
cp .env.example .env
```

### Development

```bash
npm run dev
```

### Build

```bash
npm run build
```

### Testing

```bash
# Unit tests
npm run test

# E2E tests
npm run test:e2e
```

## Project Structure

```

├── utils/         # Utility functions

```
